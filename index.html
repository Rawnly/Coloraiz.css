From the colors to the classes(without scss)

This library is ready to use without learning sass/scss, it's compiled in a single .css file with a class for every color shade. In this Guide you'll learn how to implement in your projects.

Assigning the classes

Available colors

  • Blue
  • Purple
  • Light Red
  • Orange
  • Green
  • Marine
  • Azure
  • Light Gray
  • Gray

Shades

What Coloraiz wants to do is semplify the color selection, so we added 13 shades for every colors. Shades are structured with numbers, ranging from 1 to 13 (from lighter to darker)

Here the list with classes:

  • .cz-$colorname-1
  • .cz-$colorname-2
  • .cz-$colorname-3
  • .cz-$colorname-4
  • .cz-$colorname-5
  • .cz-$colorname-6
  • .cz-$colorname-7
  • .cz-$colorname-8
  • .cz-$colorname-9
  • .cz-$colorname-10
  • .cz-$colorname-11
  • .cz-$colorname-12
  • .cz-$colorname-13


So as you can see there is a prefix before every $colorname and next there is the number of the shade.

Subclasses

All right, we have seen that you have 13 shades but.. How can i add a background color? Simple! There are 2 There is the class .cz-bg it turns the color in a background color
Backgrounds
To add the background color you only need to add the .cz-bg class next at the class of the color else if you want to color only the text you need to add only the class color.
